HiAuRo 独立静态工具

HiAuRo Web Tools

这里放的是独立运行的网页工具，不依赖宿主 WebUI 在线。轴编辑器支持手动导入本地 trigger-catalog.json，录制阅读器支持直接打开本地日志文件。

轴编辑

Tree
树形轴编辑器
执行轴 / 辅助轴的树形结构编辑，适合逐层查看节点和属性。
打开
Flow
Flow 轴编辑器
基于 Drawflow 的画布编辑形态，适合拖拽排布节点关系。
打开
Fact
事实轴编辑器
查看和编辑事实轴阶段、事件、分支以及录制导入结果。
打开

记录查看

Recorder
Combat Recorder 阅读器
读取本地 .jsonl 日志，按事件组查看差异和时间线。
打开
轴编辑器需要你先从宿主生成本地 trigger-catalog.json，然后在页面中手动导入。 这条链路不依赖 GitHub catalog，也不要求插件在线。
